Instructions, please read!

In this experiment, an audio track will play when the page loads. The track will loop continuously. There will be images and/or radio buttons on every page. The goal is for the listener to choose the image which they identify the sound with. The listener is to select one of the options presented in the buttons, and/or describe it in the textbox -- an area for the listener to write down any other comments is also presented on every page, and the user is encouraged to describe the sound to the best of their abilities. There is no right or wrong answer.

Note: The "submit" button will not activate until 10 seconds have passed. Please take the time to listen to the audio carefully and thoroughly.
