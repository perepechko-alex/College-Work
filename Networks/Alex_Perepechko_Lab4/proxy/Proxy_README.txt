To run proxt.py, open up the command-line and type in the following:

python server.py --port=####

For example, running 'python proxy.py --port 8080' will allow you to run the proxy on port 8080. With the web browser configured correcty, the terminal will return the GET and POST requests from the website, along with the website's url. Any errors that are raised (404, 501) and all responses are displayed in the terminal.
